Acerca de este Proyecto

Este proyecto fue creado por Sandra Garaycochea como parte del bootcamp SoyHenry

Tecnologías Utilizadas

  • HTML5, CSS3, JavaScript
  • Live Server para trabajo en real time
  • ManeJo directo sobre el DOM
  • AJAX
  • JQUERY
  • MongoDB y mongoose

Características del Proyecto

  • Trabajamos con un objeto conteniendo la información de las películas, la cual es mostrada en tarjetas en la pantalla. Este objeto se encuentra en un archivo js separado
  • Presenta una barra de navegación con un logo y cuatro enlaces, dos a páginas secundarias mostrando información relacionada y otro a un formulario para agregar peliculas
  • Cada página secundaria tiene su propia barra de navegación con cuatro links, uno regresa a la página principal y los otros van a las páginas secundarias y al formulario
  • Tiene una página con un formulario para agregar nuevas películas a la base de datos
